## Task 2: Background — gradiente azul–violeta

**Files:**
- Modify: `app/src/main/res/drawable/ic_launcher_background.xml`

El background del adaptive icon es recortado por el launcher a la forma del dispositivo (círculo, squircle, etc.). No necesita `<corners>` — el sistema aplica el recorte. El gradiente usa exactamente los mismos colores que `bg_balance_card.xml`.

- [ ] **Reemplazar `ic_launcher_background.xml`**

Contenido completo:

```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android">
    <gradient
        android:startColor="#1D4ED8"
        android:endColor="#7C3AED"
        android:angle="135" />
</shape>
```

- [ ] **Commit**

```bash
git add app/src/main/res/drawable/ic_launcher_background.xml
git commit -m "style: launcher icon background — blue-violet gradient"
```

## Task 3: Foreground — escudo + checkmark

**Files:**
- Modify: `app/src/main/res/drawable/ic_launcher_foreground.xml`

El canvas del foreground es 108×108dp (estándar adaptive icon). La "safe zone" es un círculo de 66dp de diámetro centrado (radio 33dp desde el centro 54,54) — el contenido dentro de esta zona nunca queda recortado en ninguna forma de launcher.

El diseño escala el escudo original del concepto B (viewport 64×64) a factor 0.9 y lo centra en el canvas 108×108:
- Fórmula X: `x_orig × 0.9 + 25.2`
- Fórmula Y: `y_orig × 0.9 + 24.3`
- Radio máximo del contenido resultante: ~31.2dp < 33dp de safe zone ✓

El escudo usa dos paths separados: uno con `fillAlpha="0.2"` para el área semitransparente, otro sin relleno (`fillColor="#00000000"`) solo con stroke para el contorno. El checkmark es stroke puro.

- [ ] **Reemplazar `ic_launcher_foreground.xml`**

Contenido completo:

```xml
<?xml version="1.0" encoding="utf-8"?>
<vector xmlns:android="http://schemas.android.com/apk/res/android"
    android:width="108dp"
    android:height="108dp"
    android:viewportWidth="108"
    android:viewportHeight="108">

    <!-- Escudo: relleno semitransparente -->
    <path
        android:fillColor="#FFFFFF"
        android:fillAlpha="0.2"
        android:pathData="M54,30 L74,39 L74,55 C74,66 65,75 54,78 C43,75 34,66 34,55 L34,39 Z" />

    <!-- Escudo: contorno blanco sólido -->
    <path
        android:fillColor="#00000000"
        android:strokeColor="#FFFFFF"
        android:strokeWidth="2.5"
        android:strokeLineCap="round"
        android:strokeLineJoin="round"
        android:pathData="M54,30 L74,39 L74,55 C74,66 65,75 54,78 C43,75 34,66 34,55 L34,39 Z" />

    <!-- Checkmark -->
    <path
        android:fillColor="#00000000"
        android:strokeColor="#FFFFFF"
        android:strokeWidth="3.5"
        android:strokeLineCap="round"
        android:strokeLineJoin="round"
        android:pathData="M45,53 L51,59 L63,47" />

</vector>
```

- [ ] **Commit**

```bash

git commit -m "style: launcher icon foreground — shield + checkmark vector"
```
